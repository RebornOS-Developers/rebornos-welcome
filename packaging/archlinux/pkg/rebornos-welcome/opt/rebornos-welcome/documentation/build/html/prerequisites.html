Prerequisites - What you need to run Fenix Installer¶

Note: Currently, Fenix Installer only supports Arch Linux and its derivatives (like RebornOS, EndeavourOS, Manjaro, etc.). Your build system (where you run Fenix Installer) should be booted into Arch Linux or one of its derivates before proceeding.

Follow one of the below sections to set up Fenix Installer before running. The below commands must be run on a terminal.

Using the Setup Script¶

  1. Download the project:

    git clone https://gitlab.com/rebornos-team/fenix/fenix-installer.git

  2. Run the setup script:

    cd fenix-installer
sh setup-archlinux.sh

If the script ran successfully, congratulations! You are ready to run Fenix Installer. Proceed to Usage for further steps.

Manual Setup¶

Warning: Please skip this section if you have already run the setup script successfully.

  1. Install python dev tools:

    • Install the packages python and pip

    sudo pacman -S --needed python python-pip

    • Install pipenv through pip

    sudo pip install pipenv

  1. Install other packages: xdg-utils and git

    sudo pacman -S --needed git xdg-utils

  2. The program itself: Download Fenix Installer into a directory of your choice

    git clone https://gitlab.com/rebornos-team/fenix/fenix-installer.git

  3. Runtime dependencies: Install the python modules required by Fenix Installer

    cd fenix-installer
pipenv install

  4. Executable permissions: Provide executable permissions to the installer script

    sudo chmod +x fenix-installer.sh

If you performed all the above steps successfully, congratulations! You are ready to run Fenix Installer. Proceed to Usage for further steps.
