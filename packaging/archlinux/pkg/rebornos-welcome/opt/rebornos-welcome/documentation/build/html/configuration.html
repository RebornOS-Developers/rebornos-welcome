Configuration - Customizing the Fenix Installer¶

Configuration of Fenix Installer is done through JSON files. They are similar to Python dict s, except that JSON uses boolean values with the first letter not capitalized (for example, true in JSON as opposed to True in Python). The configuration files are found at fenix-installer/configuration/

installer.json¶

Todo

The documentation of installer.json

Below is a brief description of currently offered configuration variables:

“build_directory”

The absolute path to the directory in which you would like archiso files to be copied before modifying them.

“work_directory”

The absolute path to the directory in which you would like mkarchiso to install all the ISO packages and copies all the package caches before writing them all to an ISO.
