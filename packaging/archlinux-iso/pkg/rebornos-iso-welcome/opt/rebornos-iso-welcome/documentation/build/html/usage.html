Usage¶

Warning: Make sure that you have read and followed the steps in Prerequisites - What you need to run Fenix Installer before proceeding.

Running - Launching Fenix Installer¶

On a terminal, ensure that you are in the project base directory (fenix-installer/) and run:

./fenix-installer.sh

Logging - Checking for progress and issues¶

After you launch Fenix Installer, you can keep track of the progress and check for any errors by monitoring the log. You could do it in the following ways:

  • On the GUI: Click on a small label at the bottom of the installer window (on the Graphical User Interface) titled Console Output to view the execution status of commands. The console output area can be extended by pulling up the bar at its top border.

OR

  • Open in an editor: Find the latest log in fenix-installer/log/ and open it in an editor.

OR

  • Print the latest messages: Run tail -f $( ls -t1 |  head -n 1 ) on a terminal after changing to the fenix-installer/log/ directory.
